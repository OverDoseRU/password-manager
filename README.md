# ![License](https://img.shields.io/badge/License-MIT-green) Password Manager

Менеджер паролей с шифрованием AES-256-GCM.


##Дисклеймер

Данный инструмент предназначен **только для личного использования**.
Пользователь несёт ответственность за соблюдение авторских прав.
Автор не несёт ответственности за использование программы.


## Возможности
- Шифрование паролей мастер-паролем
- Хеширование мастер-пароля (не хранится в открытом виде)
- Добавление, просмотр всех паролей, поиск по сайту
- Автоматическое создание файла с паролями при первом запуске
