# ![Ruby](https://img.shields.io/badge/Ruby-4.0.2-CC342D?logo=ruby) Password Manager

Менеджер паролей на Ruby с шифрованием AES-256-GCM.

## Возможности
- Шифрование паролей мастер-паролем
- Хеширование мастер-пароля (не хранится в открытом виде)
- Добавление, просмотр всех паролей, поиск по сайту
- Автоматическое создание файла с паролями при первом запуске

## Установка и запуск

1. Установите Ruby (рекомендуется версия 4.0+)
2. Склонируйте репозиторий:
   ```bash
   git clone https://github.com/ваш-логин/password-manager.git
   cd password-manager
